# Project Documentation

## Overview
This project is a personal website for **intelgigabyte9299**, a Custom ROM developer. The site serves as a platform to showcase various custom ROMs developed for the Samsung Galaxy A71 4G, including crDroid, WitAqua, and RisingOS Revived. It provides users with essential information, installation guides, and links to social media for support and updates.

## Project Structure
The project consists of the following files and directories:

- **index.html**: The main webpage that includes navigation, sections about the developer, custom ROMs, installation guides, and social media links.
- **css/style.css**: Contains styles for the website, defining layout, colors, fonts, and other visual aspects.
- **js/main.js**: Includes JavaScript functionality for interactive elements such as theme toggling, navigation menu behavior, and modal popups for changelogs.
- **img/favicon.ico**: The favicon for the website, displayed in the browser tab and bookmarks.
- **crdroid.html**: Detailed information about the crDroid custom ROM, including features, installation instructions, and download links.
- **witaqua.html**: Information about the WitAqua custom ROM, detailing its features and installation process.
- **risingosrevived.html**: Details about the RisingOS Revived custom ROM, including unique features and installation guidance.

## Usage
To use this project, simply open the `index.html` file in a web browser. The website is designed to be responsive and should work well on both desktop and mobile devices. Users can navigate through the different sections to learn more about the custom ROMs and find installation guides.
