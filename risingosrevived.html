✨ Key Features

  • Advanced UI customization: header images, clock styles, power menu themes
  • Ortus Launcher with Material You, icon packs, and freeform multitasking
  • Quick Settings tweaks: blur, transparency, landscape layout, slider styles
  • Lockscreen widgets: battery info, media controls, mini clock modules
  • Performance features: Smart 5G, Smart Pixels, strict standby policies
  • Enhanced security: app lock, hide ADB, screen capture indicator toggle
  • Audio tools: music ticker, status bar lyrics, pulse visualizer
  • File system support: exFAT, NTFS, ZRAM (unofficial builds)
  • Hardware tools: flashlight strength, refresh rate OC, GPU undervolt
  • Misc utilities: Game space, pocket mode, edge lighting, now bar
  • OTA support for updates and changelogs via built-in updater
