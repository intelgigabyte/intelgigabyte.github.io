✨ Key Features

  • Clean, bloat-free Android experience with minimal UI customization for better speed and stability.
  • Lightweight tweaks: theme picker, status bar icon options, and quick settings layout adjustments.
  • Lock screen customization: charging animation, pickup-to-wake, and optional Always-On Display (AOD).
  • Gesture navigation support, multiple navbar styles, and icon pack compatibility.
  • Notification controls: heads-up toggle, clipboard access manager, and flashlight blink on alerts.
  • Audio enhancements: adaptive playback engine and customizable volume panel interface.
  • Extra tools: Game Space, pocket mode detection, and hardware sensor toggle options.
  • Monthly OTA updates with security patches and system-level improvements.
